# Nurulla Hasan | Creative Engineer Portfolio

An architectural, high-performance portfolio built for the next generation of web engineering. This project showcases technical precision, fluid motion, and a unique "Creative Engineer" aesthetic.

## 🚀 Vision
To bridge the gap between mathematically precise backend logic and emotionally resonant digital interfaces. This portfolio is a living testament to that philosophy—combining structural robustness with state-of-the-art motion.

## 🛠 Tech Stack
- **Core:** Next.js 15+ (App Router), TypeScript, React 19
- **Styling:** Tailwind CSS v4 (Modern CSS-first approach)
- **Animation:** Framer Motion (Physics-based interactions)
- **Interactions:** Lenis (Global Smooth Scrolling)
- **Icons:** Lucide React (Minimalist vector set)
- **Deployment:** Vercel

## ✨ Key Architectural Features
- **Magnetic Orbit Branding:** A custom-engineered, coordinate-based branding system in the navbar.
- **Liquid Sidebar:** A glassmorphic, icon-led navigation system with staggered reveal animations.
- **Architectural HUD:** HUD-inspired layouts (Heads-Up Display) with scanning lasers and technical metadata.
- **Responsive Precision:** Fully optimized for all viewports with fluid typography scaling.
- **Glassmorphism 2.0:** Sophisticated use of backdrop filters and radial gradients for depth.

## 🏗 Project Structure
```text
src/
├── app/            # Next.js App Router & Global Config
├── components/
│   ├── home/       # Hero, About, Skills, Experience, Projects
│   ├── layout/     # Reusable SectionLayout & Wrappers
│   └── providers/  # Smooth Scroll, Theme Providers
└── lib/            # Utilities and shared logic
```

## 🚥 Getting Started

First, install the dependencies:
```bash
npm install
# or
pnpm install
```

Run the development server:
```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
